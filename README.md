# Heroes Are Made

`Godot 4.6.1` 기준으로 작업하는 프로젝트입니다.

## 요구 사항

- Godot Engine `4.6.1` (권장)
- Windows 환경 기준 렌더러 설정: `d3d12`

## 실행 방법

1. Godot 4.6.1을 실행합니다.
2. `Import`를 선택합니다.
3. 이 폴더의 `project.godot` 파일을 선택합니다.
4. 프로젝트를 연 뒤 `Run Project`로 실행합니다.

## 프로젝트 정보

- 프로젝트명: `Heroes Are Made`
- 기본 아이콘: `icon.svg`
- 3D 물리 엔진: `Jolt Physics`
- 렌더링 기능: `Forward Plus`

## 포함된 애드온

- `addons/GitGodot` (Git 연동 플러그인)

## 폴더 구조

```text
.
├─ addons/
│  └─ GitGodot/
├─ icon.svg
└─ project.godot
```
