# 전투 시스템 개편 방향

## 1. 코어 방어 구조 강화
- 코어 내구도 중심 방어 목표를 강화한다.
- 코어 주변 방어 지점을 활용해 압박 방향 읽기와 배치 판단을 유도한다.

## 2. 히어로 역할 전환 카드
- 히어로를 고정 역할이 아닌 전투 중 태세 전환 유닛으로 운영한다.
- 같은 조합도 카드 선택에 따라 운영 방식이 달라지도록 설계한다.

## 3. 카드 타입 이원화
- 즉시 효과 카드와 전장 규칙 변경 카드를 분리한다.
- 순간 효과와 전장 제어 효과를 함께 운용해 전술 개입감을 높인다.

## 4. 웨이브 변이 도입
- 웨이브 시작 시 적 변이를 부여한다.
- 플레이어는 보상 카드/유물로 변이에 대응하는 카운터 빌드를 구성한다.

## 5. 보상 이원화
- 보상 축을 골드와 카드/유물로 분리한다.
- 경제 성장 선택과 전술 성장 선택의 충돌을 의도적으로 만든다.
