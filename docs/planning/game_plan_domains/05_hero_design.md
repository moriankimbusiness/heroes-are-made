# 히어로 기획

## 기본 규칙
- 히어로 최대 보유 수는 4명이다.
- 전투 중 실시간 뽑기/소환 방식은 사용하지 않는다.
- 각 히어로는 종족과 클래스 조합을 가진다.
- 각 히어로는 개별 체력을 가지며, 체력 0 시 전투 불능 상태가 된다.

## 편성/합류 규칙
- 시작 모험단은 3명으로 고정한다.
- 런 시작 전 편성 단계에서 초기 모험단을 확정한다.
- 런 진행 중 신규 합류는 월드 이벤트/보상 노드로만 발생한다.

## 이동/전투 운영 규칙
- 방어 기준점은 코어 중심으로 설정한다.
- 기본 플레이 영역 반경은 220이다.
- 조작 방식은 좌클릭 선택 + 우클릭 이동 명령이다.
- 이동 명령은 선택된 히어로 1명에게만 적용한다.
- 이동 중에는 자동공격을 보류하고, 이동 종료 후 자동공격으로 복귀한다.

## 사거리 규칙
- 히어로 선택 시 실제 공격 사거리 원형을 표시한다.
- 사거리 계산은 기본값, 추가값, 배율을 조합해 산출한다.
- 사거리 갱신은 초기화, 장비 변경, 스탯 재계산, 상태 변화 시점에 즉시 반영한다.
