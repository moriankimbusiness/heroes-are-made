# 원정 및 스토리 구조

## 원정 기본 규칙
- 챕터 단위 런 구조를 사용한다.
- 챕터는 순서대로 진행하는 선형 구조를 기본으로 한다.
- 챕터 클리어 시 다음 챕터로 이동한다.
- 모험단 최대 인원은 4명으로 제한한다.

## 런 실패 규칙
- 전투 패배 시 해당 챕터 런은 즉시 종료한다.
- 실패한 노드 재도전은 허용하지 않는다.
- 메타 진행(영구 성장/영구 해금)은 현재 범위에서 제외한다.

## 챕터 준비 단계
- 준비 화면 기능은 편성과 장비 관리로 제한한다.
- 준비 단계에서는 상점/강화/회복 서비스를 제공하지 않는다.
- 출정 확정 시 월드맵 생성 후 진행을 시작한다.
