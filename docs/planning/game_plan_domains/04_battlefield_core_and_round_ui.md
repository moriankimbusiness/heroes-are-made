# 전장, 코어, 라운드 UI

## 코어(수레) 규칙
- 모험단 수레를 전투의 코어로 사용한다.
- 코어는 맵 중앙에 배치한다.
- 전투 최우선 보호 대상은 코어다.
- 코어는 개별 체력(HP)을 가지며, 상시 체력 표시 UI를 유지한다.

## 패배 조건
- 코어 체력이 0이 되면 패배한다.
- 또는 모든 히어로가 전투 불능이면 패배한다.
- 기존 통과 횟수 기반 패배 규칙은 사용하지 않는다.

## 라운드 UI 및 속도 제어
- 상단 중앙 라운드 타이머 옆에 속도 제어를 배치한다.
- 제공 속도: 일시정지, 1배속, 2배속
- 기본 시작 속도는 1배속이다.
- 속도 변경은 버튼 입력 기반으로 처리한다.

## 카메라 규칙
- 전투 카메라는 수동 이동/줌을 지원한다.
- 이동 입력: WASD
- 줌 입력: 마우스 휠
- 휠 업은 축소, 휠 다운은 확대로 통일한다.
