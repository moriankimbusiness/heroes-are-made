# 런 진행 플로우 및 월드맵

## 기본 진행 루프
- 메인화면 → 챕터 시작 준비 → 게임월드(경로 선택) → 노드 해소 → 노드 결과 요약 → 게임월드 복귀
- 챕터 최종보스 클리어 시 챕터 결과 화면으로 이동한다.
- 전투 패배 시 런 실패 결과 화면으로 이동한다.

## 월드맵 생성 규칙
- 시작 노드 1개, 최종 노드 1개를 고정한다.
- 중간 깊이 구간은 깊이별 1~3개 노드를 랜덤 배치한다.
- 월드맵 정보(노드 타입/연결)는 챕터 시작 시 전체 공개한다.

## 깊이 규칙
- D0: 시작 노드 1개
- D1~D6: 깊이별 1~3개 랜덤
- D7: 최종보스 노드 1개

## 노드 타입 배치 규칙
- D4에 중간보스 노드를 정확히 1개 배치한다.
- D7은 최종보스 노드로 고정한다.
- 마을 노드는 D2~D5 구간에서 챕터당 최소 1회 보장한다.
- 동일 깊이에서 노드가 3개일 때 동일 타입 3중복은 금지한다.

## 경로 연결 규칙
- 다음 깊이의 인접 칸 중심으로 연결한다.
- 모든 노드는 최소 1개 진입 경로와 최소 1개 진출 경로를 가진다.
- 고립 노드는 생성하지 않는다.

## 노드 타입 정의
- 일반 전투 노드
- 중간보스 노드
- 최종보스 노드
- 아이템 획득 노드
- 마을 노드
- 이벤트 노드

## 노드 해소 규칙
- 전투 계열 노드는 전투 화면으로 전환한다.
- 아이템 획득 노드는 3개 제시 후 1개 선택한다.
- 마을 노드는 회복/강화/상점/정비 흐름으로 처리한다.
- 이벤트 노드는 선택지 결과를 즉시 확정한다.
- 모든 노드는 해소 후 결과 요약을 거쳐 월드맵으로 복귀한다.

## 저장/복구 정책
- 노드 선택 확정 직후 챕터 상태를 저장한다.
- 노드 보상 선택 확정 직후 런 상태를 저장한다.
- 선택 미확정 상태는 저장 반영하지 않는다.
